# 💇‍♀️ SalonBook – Online Salon Appointment System

SalonBook is a simple, fully client-side web application that allows users to browse salon services, book appointments, and manage their booking history — all stored locally in the browser.

This project is ideal for mini projects, academic submissions, and frontend development practice. No backend or database is required.

---

## 🚀 Features

- 🧭 Multi-page layout:
  - Home
  - Services
  - Book Appointment
  - Appointment History

- 💆 Service catalog with images and pricing  
- ⚡ One-click "Book Now" from services page  
- 📅 Appointment booking with date & time-slot selection  
- 💾 LocalStorage-based data persistence  

### 📜 Appointment Management
- View all appointments  
- ❌ Cancel appointments  
- 🔄 Reschedule appointments (modal)  
- 🔁 Book again (pre-filled form)  

### 🌙 Theme Support
- Dark / Light mode toggle  
- Theme stored in LocalStorage  
- Persistent across all pages  

### 📱 UI & Design
- Responsive layout using Bootstrap 5  
- Clean and modern UI  
- Bootstrap Icons used  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Bootstrap 5  
- Bootstrap Icons  
- Browser LocalStorage  

---

## 🧭 Pages Overview

### 🏠 Home (`index.html`)
- Introduction and landing section  
- Call-to-action buttons  
- Feature highlights  

### 💆 Services (`services.html`)
- Displays available services  
- Shows images and pricing  
- "Book Now" redirects with selected service  

### 📅 Book Appointment (`booking.html`)
- Booking form  
- Select service, date, and time  
- Saves appointment to LocalStorage  

### 📜 Appointment History (`history.html`)
- Displays all appointments  
- Cancel appointments  
- Reschedule via modal  
- Book again with pre-filled data  

---

## 🌙 Dark Mode Support

- Toggle between Light and Dark themes  
- Preference saved in LocalStorage  
- Automatically applied across all pages
