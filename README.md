# 💇‍♀️ SalonBook – Online Salon Appointment System

SalonBook is a simple, fully client-side web application that allows users to browse salon services, book appointments, and manage their booking history — all stored locally in the browser.

This project is ideal for mini projects, academic submissions, and frontend development practice. It does not require any backend or database.

---

## 🚀 Features

- 🧭 Multi-page layout:
  - Home
  - Services
  - Book Appointment
  - Appointment History

- 💆 Service catalog with images and pricing  
- ⚡ One-click **"Book Now"** from Services page  
- 📅 Appointment booking with date & time-slot selection  
- 💾 LocalStorage-based data persistence (no backend required)  

### 📜 Appointment Management
- View all appointments  
- ❌ Cancel appointments  
- 🔄 Reschedule appointments (via modal)  
- 🔁 Book again (auto pre-fill form)  

### 🌙 Theme Support
- Dark / Light mode toggle  
- Theme preference saved in LocalStorage  
- Consistent theme across all pages  

### 📱 UI & Design
- Responsive layout using Bootstrap 5  
- Clean and modern UI  
- Bootstrap Icons integration

## 🧭 Pages Overview

### 🏠 Home (`index.html`)
- Landing page with introduction  
- Call-to-action buttons  
- Highlights key features  

---

### 💆 Services (`services.html`)
- Displays available salon services  
- Includes images and pricing  
- "Book Now" redirects to booking page with selected service  

---

### 📅 Book Appointment (`booking.html`)
- Appointment booking form  
- Select service, date, and time slot  
- Stores data in LocalStorage  

---

### 📜 Appointment History (`history.html`)
- Displays all booked appointments  
- Cancel appointments  
- Reschedule appointments (modal popup)  
- Book again with pre-filled details  

---

## 🌙 Dark Mode Support

- Toggle between Light and Dark themes  
- Preference stored using LocalStorage  
- Automatically applied across all pages
